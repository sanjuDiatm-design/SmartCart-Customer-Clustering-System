# 🛒 SmartCart Customer Clustering System

## 📌 Project Overview

The SmartCart Customer Clustering System is a machine learning project designed to segment customers into distinct groups based on their purchasing behavior and shopping patterns.

Using unsupervised learning techniques, this project helps businesses better understand customer behavior, improve marketing strategies, personalize recommendations, and enhance customer retention.

---

## 🎯 Objectives

- Analyze customer shopping behavior.
- Perform data cleaning and preprocessing.
- Identify meaningful customer segments.
- Apply clustering algorithms for customer segmentation.
- Visualize customer groups for business insights.

---

## 📂 Dataset

The dataset contains customer-related information such as:

- Customer ID
- Age
- Gender
- Annual Income
- Spending Score
- Purchase Frequency
- Other shopping-related features

> Dataset used: `smartcart_customers.csv`

---

---

## 🔄 Project Workflow

### 1. Data Collection
- Load customer dataset.
- Inspect data structure and quality.

### 2. Data Preprocessing
- Handle missing values.
- Remove duplicates.
- Feature scaling and transformation.

### 3. Exploratory Data Analysis (EDA)
- Understand customer demographics.
- Analyze spending patterns.
- Visualize feature distributions.

### 4. Customer Clustering
- Determine optimal number of clusters.
- Apply K-Means Clustering.
- Evaluate cluster performance.

### 5. Visualization
- Cluster distribution plots.
- Customer segment analysis.
- Business interpretation of results.

---

## 📊 Results

The model successfully segmented customers into multiple groups based on shopping behavior and spending characteristics.

Example customer segments:

- High Income – High Spending
- High Income – Low Spending
- Moderate Income – Moderate Spending
- Budget-Conscious Customers
- Premium Customers

These insights can be used for:

- Personalized marketing campaigns
- Product recommendations
- Customer retention strategies
- Revenue optimization
